Projects & Evidence

01
Arduino DHT22 I2C LCD Alert System
IoT Sensor Monitor with Alert System

Real-time temperature and humidity monitor with a three-state visual and audio alert system. DHT22 readings displayed on a 16×2 I2C LCD; a green/yellow/red LED status model reacts to threshold logic (35°C / 80% RH triggers buzzer). Shows hardware-software co-design: sensor polling, display management, and conditional I/O in a single loop.

Live Simulation
Wokwi · Arduino
02
ESP32 HC-SR04 Ultrasonic Distance Sensing
Smart Parking Slot Detector

Proximity-based parking detection using HC-SR04 ultrasonic ranging on ESP32. Generates 10µs trigger pulses, times the echo return, converts to centimetres, and triggers a red/green indicator with occupancy status over Serial — refreshed every second. Demonstrates timed pulse I/O and real-time decision logic at the hardware level.

Live Simulation
Wokwi · ESP32
03
Arduino LDR Analog Input Control Loop
Automated Street Light System

LDR-based closed-loop ambient light control: reads ADC voltage from an LDR voltage divider, compares against a calibrated threshold (600), and switches the LED automatically. A clean implementation of analog sensing → digital decision, the foundation of every reactive embedded system.

Live Simulation
Wokwi · Arduino
